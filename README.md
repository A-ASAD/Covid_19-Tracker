# COVID-19 TRACKER
Basic app to show covid data around the globe numerically and graphically.

## FEATURES
- Check covid data from a list of 180+ countries
- Interactive graphs for data representation

## TECH STACK
- React Js
- Node js
- ChartJS
- HTML
- CSS
